_**Question**_

Find the g.l.b for the following sets.

(a) $$(7, 8)$$

(b) $$\{\pi + 1, \pi + 2, \pi + 3, \ldots\}$$

(c) $$\{\pi + 1, \pi + \dfrac{1}{2}, \pi + \dfrac{1}{3}, \pi + \dfrac{1}{4}, \ldots\}$$

[//]: # _**Intuition**_

_**Ans**_

(a) $$7$$

(b) $$\pi + 1$$

(c) $$\pi$$
